# MOOd

A daily mood-sharing social feed — implemented from the Wonder design project.

## Pages

- **/** — Gallery feed (iridescent theme)
- **/themes** — Daily theme post composer
- **/grounded** — Gallery feed (grounded/pastel theme variant)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- [Next.js](https://nextjs.org/) 15 (App Router)
- [React](https://react.dev/) 19
- [Tailwind CSS](https://tailwindcss.com/) 4
- [react-icons](https://react-icons.github.io/react-icons/)

## Design Source

UI designs were extracted from the Wonder canvas project "Welcome to Wonder" via the Wonder MCP server.
